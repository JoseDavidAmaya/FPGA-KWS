This demonstration identifies one of 10 words in an audio recording using a neural network running in an FPGA

Instructions:

  1. Click on Record from microphone button on the left
    • If it's the first time using this demo, a popup will ask for permission to use the microphone, click on Allow
  2. Say one of the supported words (shown below) and then click on Stop recording
  3. Click on Submit
  4. To try more words, click on Clear and start from step 1
